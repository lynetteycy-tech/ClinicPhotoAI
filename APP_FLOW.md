# App Flow

## Screens in Order
1. Login Screen
2. Sign Up Screen (for new clinics)
3. Patient Search Screen
4. Add Patient Form
5. Consent Signature Screen
6. Capture Instructions
7. Angle 1: Front (5s recording)
8. Review Single Photo (Accept/Reject)
9. Angle 2: Left 45° (5s recording)
10. Review Single Photo (Accept/Reject)
11. Angle 3: Left 90° (5s recording)
12. Review Single Photo (Accept/Reject)
13. Angle 4: Right 45° (5s recording)
14. Review Single Photo (Accept/Reject)
15. Angle 5: Right 90° (5s recording)
16. Review Single Photo (Accept/Reject)
17. All Photos Grid Review
18. Save Session
19. Patient Gallery

## Key User Actions
- **Search patients** - Search by name, IC, or registration number
- **Add patient** - Enter details, auto-generate registration number
- **Sign consent** - Draw signature on screen
- **Start capture** - Begin 5-angle sequence
- **Retake** - Reject photo and retake same angle
- **Export** - Save photos to device gallery
